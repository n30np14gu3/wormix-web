# wormix-web

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

## Default .env fle
```dotenv
VUE_APP_API_BASE="http://127.0.0.1:8000/api/"
VUE_GAME_URL="http://127.0.0.1:3137/preloader.html"
```
